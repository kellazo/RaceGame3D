A 3D Race Game

This game consist of to take 10 balls before 20 seconds!

If you take all the balls, then your time is save for next game.

Try it!

Instructions:

Arrow UP : Accelerate
Arrow Down: Back
Arrow Left/Right: Direction of car

Keys:

R: Restar game

F1: Debug mode
F2: God mode
C: Create a ball
